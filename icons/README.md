Drop `.ico` files here and `just build` will bake them into the desktop launcher.

Use filenames based on the launcher title:

- `terminal.ico`
- `clock.ico`
- `system-info.ico`
- `hello-app.ico`
- `calculator.ico`
- `notepad.ico`
- `paint.ico`

The build normalizes names to lowercase with hyphens, so spaces and punctuation in app titles become `-`.
